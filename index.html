Confira abaixo o conceito das Tags e Atributos no html

O que são e para que servem as tags?

Tags são os atalhos de comandos utilizados no html para configurar a página da web em questão. É uma maneira de definir as condições dos textos, imagens e outras informações presentes na página.

  • Head: É o comando dado para inserir a primeira coisa que o usuário/client irá ver na pagina;
  • Body: Tudo o que é escrito entre esse parametro body. Será o conteúdo da página;
  • h1: Serve para inserir o primeiro texto da página ou o título;
  • p: Comando utilizado para por o paragrafo;
  • u: Pode ser utilizado para sublinhar a escrita;
  • i: Uitilizado para que o texto fique em itálico;

E os atributos são utilizados quando?

Os atributos são algumas caracteristas que algumas tags podem ter. Isso quer dizer que não são todas as tags que possuema algum tipo de atributo. Como por exemplo identificar com o texto ou palavra e até direcionar a um link. Abaixo você vai encontrar 3 tipos de atributos:

  1. Id: Serve para identificar um elemento.
  2. Style: Tem o objetivo de custominar. No CSS ou no html é possivel inserir cores e etc.
  3. Class: É uma forma de classificar todas as tags com determinada nome, essa classificação pode ocorrer por cores.
